- [tfsec](https://github.com/aquasecurity/tfsec) - Static analysis security scanner for Terraform code
- [Checkov](https://github.com/bridgecrewio/checkov) - Static code analysis tool for infrastructure-as-code
- [KICS (Keeping Infrastructure as Code Secure)](https://github.com/Checkmarx/kics) - Security vulnerability scanning and compliance testing
- [Terrascan](https://github.com/tenable/terrascan) - Static code analyzer for Infrastructure as Code
- [tflint](https://github.com/terraform-linters/tflint) - Terraform linter focused on possible errors and best practices
- [Terratest](https://github.com/gruntwork-io/terratest) - Testing infrastructure code
- [terraform-compliance](https://terraform-compliance.com/) - Compliance as code framework
- [config-lint](https://github.com/stelligent/config-lint) - Rules-based configuration validation
- [Infracost](https://github.com/infracost/infracost) - Shows cloud cost estimates for Terraform
- [Terragrunt](https://github.com/gruntwork-io/terragrunt) - Thin wrapper for Terraform that provides extra tools
- [Atlantis](https://github.com/runatlantis/atlantis) - Terraform pull request automation
- [driftctl](https://github.com/snyk/driftctl) - Infrastructure drift detection tool
- [OTS (Open Terraforming Server)](https://github.com/leg100/otf) - Open source alternative to Terraform Enterprise
- [Terraformer](https://github.com/GoogleCloudPlatform/terraformer) - Infrastructure to code tool (reverse Terraform)
- [CDK for Terraform](https://github.com/hashicorp/terraform-cdk) - Cloud Development Kit for Terraform
- [Former2](https://github.com/iann0036/former2) - Generates Terraform/CloudFormation/Troposphere configurations from existing AWS resources
- [cf2tf](https://github.com/DontShaveTheYak/cf2tf) - CloudFormation to Terraform converter
- [cf-to-tf](https://github.com/humanmade/cf-to-tf) - CloudFormation to Terraform converter
- [specctl](https://github.com/aws/amazon-ecs-cli-v2) - Kubernetes to ECS converter
- [hcl2json](https://github.com/tmccombs/hcl2json) - HCL to JSON converter
- [terraform-docs](https://github.com/terraform-docs/terraform-docs) - Documentation generator for Terraform modules
- [Brainboard](https://www.brainboard.co) - Visual infrastructure builder and Terraform code generator
- [AirIAM](https://github.com/bridgecrewio/AirIAM) - AWS IAM least privilege Terraform framework
- [Pike](https://github.com/jameswoolfenden/pike) - IAM policy analysis tool
- [Terraform AWS Secure Baseline](https://github.com/nozaq/terraform-aws-secure-baseline) - AWS account secure baseline configuration
- [Terragrunt example infrastructure](https://github.com/gruntwork-io/terragrunt-infrastructure-live-example) - Example implementation of Terragrunt
- [yor](https://github.com/bridgecrewio/yor) - Automated infrastructure tagging tool
- [tfmigrator CLI](https://github.com/tfmigrator/cli) - Terraform configuration and state migration tool
- [tfgen](https://github.com/refl3ction/tfgen) - Terraform monorepo management tool
- [tfquery](https://github.com/mazen160/tfquery) - SQL queries for Terraform infrastructure
- [tfnotify](https://github.com/mercari/tfnotify) - Terraform execution notification tool
- [Conftest/Confectionery](https://github.com/open-policy-agent/conftest) - Policy testing against configurations
